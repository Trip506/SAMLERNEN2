#------------------------------------------
# Installation:
#------------------------------------------

 1. Install standard paket Contao in your domain.
    (ex. extract ZIP and FTP upload to your webspace)
    Don't use the InstallTool now, but as point 5!

 2. Installation of all listed extensions.

 3. Save this website-template in the path /templates/.

 4. Save all files needed for the website-template in path
    /templates/ and /files/ or subdirectories (just like
    in the template creation).

 5. Run the InstallTool of Contao now and import from
    the website-template.
